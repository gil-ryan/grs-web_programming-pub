## HTTP Status Codes

Check out a comprehensive list [here](https://www.restapitutorial.com/httpstatuscodes.html)!

Otherwise, here's a short table of the most common ones for a quick, convenient reference.

| Code | Purpose |
|:-:|:-:|
|200| OK|
|201|Created|
|204|No Content|
|304|Not Modified|
|400|Bad Request|
|401|Unauthorized|
|403|Forbiddnen|
|404|Not Found|
|409|Conflict|
|500|Internal Server Error|
